# 차세대 AI 에이전트 아키텍처 트렌드

최근 AI 에이전트 개발 트렌드가 '단일 프롬프트' 방식에서 'Agentic Workflow' 방식으로 전환되고 있습니다.
Andrew Ng 교수가 언급했듯이, 모델 자체의 성능 향상보다 에이전트의 작업 흐름(Reflection, Tool Use, Planning, Multi-agent collaboration)을 설계하는 것이 더 큰 성능 향상을 가져옵니다.

특히 주목할 점은 다음과 같습니다:
1. **Reflection**: 자신이 생성한 코드를 다시 검토하고 수정하는 과정.
2. **Tool Use**: 검색엔진, 계산기, 코드 인터프리터 등 외부 도구를 적절히 호출.
3. **Planning**: 복잡한 작업을 하위 작업으로 분해.

이러한 패턴을 적용하면 GPT-3.5 수준의 모델로도 GPT-4에 준하는 문제 해결 능력을 보여줄 수 있다는 연구 결과가 있습니다.
